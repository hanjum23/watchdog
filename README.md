# Watchdog — Free Website Monitor

A clean, real-time website uptime monitoring dashboard built with React.

## Deploy in 2 minutes

### Option A: Netlify (drag & drop — no CLI needed)
1. Run `npm install && npm run build` locally
2. Go to [netlify.com/drop](https://app.netlify.com/drop)
3. Drag the `build/` folder onto the page
4. Done — you get a live URL instantly

### Option B: Vercel (one command)
```bash
npm install -g vercel
npm install
vercel --prod
```

### Option C: GitHub + Netlify/Vercel (auto-deploys)
1. Push this folder to a GitHub repo
2. Connect the repo in Netlify or Vercel dashboard
3. Every push auto-deploys

## Run locally
```bash
npm install
npm start
```
Opens at http://localhost:3000
